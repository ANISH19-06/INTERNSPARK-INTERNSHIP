# 8-bit ALU RTL Design, Synthesis and Gate-Level Simulation using Verilog and Xilinx Vivado

## Overview

This project demonstrates the complete digital design flow of an 8-bit Arithmetic Logic Unit (ALU), including RTL design, functional verification, synthesis, gate-level netlist generation, and gate-level simulation using Xilinx Vivado.

---

## Features

- 8-bit ALU Design
- Arithmetic Operations
  - Addition
  - Subtraction
- Logical Operations
  - AND
  - OR
  - XOR
  - NOT
- Shift Operations
  - Left Shift
  - Right Shift
- Status Flags
  - Carry Flag
  - Zero Flag
  - Overflow Flag

---

## Tools Used

- Verilog HDL
- Xilinx Vivado
- Vivado Simulator

---

---

## Files Included

- alu.v : RTL design of 8-bit ALU
- alu_tb.v : RTL testbench
- alu_gate_tb.v : Gate-level testbench
- alu_netlist.v : Synthesized gate-level netlist
- Reports : Utilization and timing reports
- Simulation screenshots

---

## Results

The synthesized gate-level implementation produced outputs equivalent to the RTL design for all arithmetic, logical, and flag operations.
