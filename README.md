# InternSpark Internship - VLSI Design Tasks

## Overview

This repository contains all the tasks completed as part of my InternSpark VLSI Internship. The projects demonstrate my understanding of digital design, Verilog HDL, RTL development, functional verification, synthesis, and gate-level simulation.

## Completed Tasks

### Task 1: 8-bit ALU RTL Design and Functional Verification

- Designed an 8-bit Arithmetic Logic Unit using Verilog HDL.
- Implemented arithmetic, logical, and shift operations.
- Added Carry, Zero, and Overflow status flags.
- Developed directed and random testbenches.
- Verified functionality using simulation waveforms.

### Task 2: 8-bit ALU Synthesis and Gate-Level Simulation

- Performed RTL synthesis using Xilinx Vivado.
- Generated a synthesized gate-level netlist.
- Analyzed utilization and timing reports.
- Developed a gate-level testbench.
- Verified RTL and gate-level output equivalence.

## Tools Used

- Verilog HDL
- Xilinx Vivado
- Vivado Simulator
- Icarus Verilog
- GTKWave
